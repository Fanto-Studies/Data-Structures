## Initial Configuration

- Use `.NET 10` to run the files.
- Install [.NET 10](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)!
- You can run the files in terminal with the following command:

```shell
dotnet run <fileName>.cs
```
